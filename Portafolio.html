Sobre mi

Desde el año 2024, comencé a probar suerte como desarrollador web, sumergiéndome en este campo con pasión y dedicación. Ahora, con un año de experiencia a mis espaldas, he desarrollado un sólido conjunto de habilidades que me permiten crear soluciones digitales eficientes y atractivas, abarcando tanto el frontend como el backend. Mi enfoque se centrarme en crear experiencias digitales de alta calidad, asegurando que cada proyecto refleje el compromiso con la perfección y la accesibilidad.

Proyectos

    Plataforma de Gestión de Tareas: Desarrollé una aplicación web que permite a los usuarios gestionar sus tareas de manera eficiente. Utilicé JavaScript para el frontend, creando una interfaz interactiva, y SQL en el backend para la gestión de la base de datos
    Tienda en Línea: Creé una tienda en línea con funciones de carrito de compras y procesamiento de pagos. El proyecto se desarrolló utilizando JavaScript para el frontend y SQL para manejar la base de datos en el backend.
    Red Social para Aficionados al Deporte: Diseñé una red social que conecta a aficionados del deporte, permitiendo la creación de perfiles y publicaciones. Utilicé Java para la lógica del backend y SQL para el almacenamiento de datos, con JavaScript manejando la interacción en el frontend.


Experiencia

    • 1 año de experiencia en el desarrollo web, comenzando en 2024. Durante este año, he trabajado en el diseño y la implementación de aplicaciones web completas, adquiriendo habilidades en el desarrollo tanto del frontend como del backend.
    • 3 proyectos realizados como desarrollador fullstack. He completado tres proyectos que abarcan diferentes áreas del desarrollo web, incluyendo la gestión de tareas, una tienda en línea y una red social, utilizando tecnologías como JavaScript, SQL y Java.
    • Desarrollo de aplicaciones web con un enfoque en la calidad y la accesibilidad. Me he especializado en crear experiencias digitales que no solo son funcionales y estéticamente agradables, sino también accesibles para una audiencia amplia.
